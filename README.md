# rimas.vim

Diccionario de rimas para Vim.
Esta es una herramienta muy util para los poetas mallditos como yo.

## Dependencias

https://github.com/xerron/merimas

Debe estar en el path del sistema.

## Uso

Mostrar rimas para una palabra:

    :Rima <palabra>

Cerrar la ventana:

    :Rima!
